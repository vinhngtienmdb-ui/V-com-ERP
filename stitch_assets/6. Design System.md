# VComm Enterprise Core Design System Specification

## Brand & Style

This design system is engineered for **VComm**, a high-scale Enterprise ERP environment. The brand personality is rooted in **Reliability, Efficiency, and Intelligence**. It targets power users who manage complex supply chains, financial audits, and multi-channel commerce operations.

The visual style follows a **Corporate / Modern** aesthetic with a strong emphasis on **Information Density**. 
- **Modern Minimalism:** We utilize clean whitespace to separate high-density data modules, ensuring the UI never feels cluttered despite the massive amounts of information.
- **Precision Engineering:** Every element is aligned to a strict grid, reflecting the logical and structured nature of ERP workflows.
- **Subtle Depth:** Instead of heavy borders, the system uses soft tonal layering and subtle shadows to define hierarchy, keeping the interface feeling lightweight and breathable.

## Layout & Spacing

The design system employs a **Fixed-Fluid Hybrid Grid**. 
- **The Sidebar:** A fixed width of 260px houses the extensive module navigation. 
- **The Content Area:** A fluid 12-column grid that expands up to a maximum width of 1600px to ensure data visualizations don't become overly stretched on ultrawide monitors.

**Sidebar Configuration (Preserved Order):**
The sidebar is categorized into logical clusters:
1. **Tổng Quan:** Siêu thị VComm, Quản lý Đơn hàng, Quản lý Livestream, Mạng xã hội.
2. **Kinh Doanh:** Quản lý sản phẩm, Marketing & Social, Flash Sale, KOL/KOC.
3. **CRM:** Khách hàng thân thiết, Quản lý Quảng cáo.
4. **Logistics:** Quản trị Kho vận, Vận chuyển, Mua hàng, Tuân thủ.
5. **Back Office:** Tài chính - Kế toán, Đối soát, Cấu hình.

**Responsive Behavior:**
- **Desktop:** 12 columns, 24px margins.
- **Tablet:** 8 columns, sidebar collapses into an icon-only rail (72px).
- **Mobile:** 4 columns, sidebar becomes a hidden drawer.

## Elevation & Depth

This design system avoids heavy drop shadows in favor of **Tonal Elevation** and **Thin Outlines**. 

- **Level 0 (Background):** The page background is `#F8FAFC`, providing a neutral canvas.
- **Level 1 (Cards/Tables):** White surfaces with a 1px solid border in `#E2E8F0`.
- **Level 2 (Interactive/Hover):** A very soft shadow (`0px 4px 12px rgba(0, 0, 0, 0.05)`) is applied only when a card or row is hovered to indicate interactivity.
- **Floating Elements:** Modals and dropdowns use a sharp, medium-diffusion shadow to separate them from the primary workspace.

This approach maintains the "clean whitespace" look while providing clear visual affordances for interactive elements.

## Components

### Data Tables
The core of the ERP. Tables must support:
- **Sticky Headers:** Always visible during scroll.
- **Compact Density:** 32px row heights for "High Density" view, 44px for "Standard".
- **Cell Alignment:** Numbers are right-aligned; text is left-aligned.

### Cards with Stats
Used for dashboard overviews.
- Must include a primary metric, a trend indicator (percentage change), and a small Sparkline chart showing the 7-day trend.

### Sidebar Navigation
- **Active State:** A vertical bar of Professional Blue on the left edge of the menu item with a light blue background tint (#F0F4F9).
- **Icons:** 20px stroke-based icons with consistent weights.

### Buttons
- **Primary:** Solid Professional Blue with white text.
- **Secondary:** Transparent background with a 1px border of `#CBD5E1`.
- **Ghost:** No border or background, used for secondary actions in tables.

### Input Fields
- Clear labels (Label-xs) always positioned above the field.
- Active state uses a 2px Professional Blue focus ring with 0% offset.
